📁 Project Setup
Step 1

ZIP extract করো ও VS Code-এ open করো

Download করা medfind_COMPLETE_v4.zip extract করো → VS Code → File → Open Folder → medfind_FINAL_FIXED folder select করো

2
🐍 Backend Setup (Django)
Step 2 — Virtual Environment

Python venv তৈরি করো

cd backend
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate   # Mac/Linux
✅ Terminal-এ (venv) দেখাবে
Step 3 — Dependencies

Packages install করো

pip install -r requirements_render.txt
✅ "Successfully installed" দেখাবে (2-3 মিনিট লাগবে)
Step 4 — .env Config ⚠️ IMPORTANT

backend/.env file already আছে — শুধু edit করো

VS Code-এ backend/.env file খোলো এবং এই values দাও:

GEMINI_API_KEY=AIzaSyAJGKkaMB4ixVPmd5r-cRR1e5ea4AxKmwQ
EMAIL_HOST_USER=তোমার_gmail@gmail.com
EMAIL_HOST_PASSWORD=xxxx-xxxx-xxxx-xxxx
⚠️ Gmail App Password: Gmail → Account → Security → 2-Step Verification → App Passwords → Create → Copy 16-digit code
Step 5 — Database

Database migrate করো (OTP + AI tables তৈরি হবে)

python manage.py migrate --settings=medfind_project.settings
✅ "Applying otp_auth.0001_initial... OK" দেখাবে
Step 6 — Run

Backend চালু করো

python manage.py runserver --settings=medfind_project.settings
✅ "Starting development server at http://127.0.0.1:8000/"
